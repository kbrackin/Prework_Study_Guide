HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • * means you want the style to effect the entirety of the HTML.
  • Use an element, such as header, body, ul, and h1, to change the styling of all of the elements of that type.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we're currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • git add -A: adds modifications in the current working branch to the staging area
  • git commit -m message: commits changes we've made to GitHub with a message of what we've done
  • git pull origin main: used to receive the branch's modifications into the local environment
  • git push origin branch-name: push changes we've made locally to our remote GitHub branch

JavaScript

  • Link a script to an HTML file using "script src=[Source_Location]" in angle brackets.
  • The script is linked in the body tags, not the header.
  • An array is used to hold a group of data. It can hold strings, integers, booleans, and so on.
  • JavaScript arrays are zero indexed, which means that the first value in the array would be indexed 0.
  • 'For' loops have three parts: The keyword 'for', a three part conditional statement, and a code block.
  • To comment out lines of code that you do not need now, use /* at the start and */ at the end of the code.
